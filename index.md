# 电商操盘手全栈技能路线图

E-Commerce Operator Full-Stack Skill Roadmap

从新手到战略家，5大阶段系统化成长路径。找到你的起点，规划你的学习之路。

完整内容请查看 [GitHub README](https://github.com/kangise/CBEC-AI-Hub)。

---

## 路线图总览

| 阶段 | 内容 | 适合等级 |
|------|------|----------|
| Stage 1: 基础认知 | 电商生态 / 平台规则 / 商业模式 / 基础财务 / 法规入门 | L1 新手 |
| Stage 2: 运营实战 | 选品 / Listing优化 / PPC广告 / 库存物流 / 客服售后 / 合规 | L1-L2 |
| Stage 3: 增长进阶 | 品牌建设 / 站外引流 / 多渠道扩展 / 用户运营 / 新市场拓展 | L2-L3 |
| Stage 4: 数据驱动 | 数据分析 / Excel / BI / A/B测试 / Python / 爬虫 / 自动化 / AI工具 | L3-L4 |
| Stage 5: 战略领导力 | 团队管理 / 供应链战略 / 资本运作 / 全球化布局 / 行业影响力 | L4-L5 |

## 水平等级

| 等级 | 称号 | 典型画像 | 年限参考 |
|------|------|----------|----------|
| L1 | 新手 Rookie | 刚入行，对电商有兴趣但缺乏实操 | 0-6个月 |
| L2 | 执行者 Executor | 能独立运营1-2个店铺/品牌 | 6月-2年 |
| L3 | 操盘手 Operator | 能独立操盘一个品牌的全盘运营 | 2-4年 |
| L4 | 专家 Expert | 数据驱动决策，能带小团队 | 4-6年 |
| L5 | 战略家 Strategist | 操盘多品牌/多市场，战略级决策 | 6年+ |

## 核心特点

- **体系化** -- 5大阶段，覆盖从基础认知到战略领导力的完整技能树
- **AI 融合** -- AI 工具和方法融入每个阶段，不是单独的章节
- **分级定位** -- 自我评估问卷帮你精准匹配当前阶段
- **资源丰富** -- Coursera、YouTube、书籍、工具，每个模块配备多元学习资源
- **实战导向** -- 每个阶段都有实践项目和能力检验标准
