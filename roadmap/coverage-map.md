## 差异化类别分析 | Differentiation Categories

以下 AI 应用类别是 CBEC-AI-Hub 独有的规划方向，竞品仓库几乎未涉及：

The following AI application categories are unique to CBEC-AI-Hub's roadmap — competing repos have minimal or no coverage:

### 🆕 1. Review 分析（NLP 驱动）

**竞品覆盖**: 无系统化覆盖。现有 "awesome" 列表仅收录通用 NLP 工具链接，未提供跨境电商场景下的 Review 分析方法论。

**CBEC-AI-Hub 规划**:
- 基于 NLP 的多语言 Review 情感分析与痛点提取
- 竞品 Review 对比分析 Prompt 模板
- Review 数据自动采集与结构化处理 Pipeline
- 差评预警与自动分类系统设计

**对应路径**: Path A（运营人）+ Path B（技术人）

### 🆕 2. 定价策略（ML 驱动）

**竞品覆盖**: 无。现有开源项目聚焦于通用定价算法，未针对跨境电商多市场、多币种、关税叠加的复杂场景。

**CBEC-AI-Hub 规划**:
- 基于 ML 的动态定价模型（考虑汇率、关税、物流成本）
- 竞品价格监控与自动调价策略
- 促销定价优化（BFCM、Prime Day 等大促场景）
- 多站点价格一致性管理

**对应路径**: Path B（技术人）+ Path C（管理者）

### 🆕 3. 社交媒体营销（AI 驱动）

**竞品覆盖**: 极少。现有仓库偏重站内运营（Amazon SEO、PPC），几乎不涉及站外社交媒体与 AI 结合的内容。

**CBEC-AI-Hub 规划**:
- AI 生成社交媒体内容（TikTok、Instagram、YouTube）
- KOL/KOC 匹配与合作效果预测
- 社交媒体评论情感监控
- 跨平台内容分发自动化

**对应路径**: Path A（运营人）

## 平台扩展路线 | Platform Expansion Roadmap

当前内容以 Amazon 为核心。多平台扩展计划如下：

| 平台 | 优先级 | 计划阶段 | 说明 |
|------|--------|----------|------|
| Amazon | 🔴 最高 | Phase 1-2 | 核心平台，当前所有内容基于 Amazon 场景 |
| Shopify | 🟡 中等 | Phase 3 | 独立站场景，与 Amazon 互补 |
| TikTok Shop | 🟡 中等 | Phase 3 | 新兴渠道，社交电商 AI 应用潜力大 |
| eBay | 🟢 较低 | Phase 3+ | 成熟平台，AI 应用场景与 Amazon 类似 |
| Walmart | 🟢 较低 | Phase 3+ | 美国市场补充，API 生态逐步完善 |
