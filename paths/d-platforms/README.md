# Path D: 多平台 AI 实战 — 超越 Amazon

> 最后更新: 2026-03-12

## 概述

Path A-C 的内容以 Amazon 为核心场景。但跨境电商不只有 Amazon — Shopify 独立站、TikTok Shop 社交电商、eBay、Walmart 等平台各有独特的 AI 应用场景。

本路径帮你把 AI 能力从 Amazon 扩展到其他平台，重点关注每个平台的差异化 AI 应用。

> 💡 **建议先修**：先完成 [Path 0 基础](../0-foundations/) 和 [Path A 运营](../a-operators/) 的核心模块，再来看平台差异。通用的 AI 方法论（Prompt 工程、Review 分析、Listing 优化）是跨平台的，本路径只聚焦差异。

## 平台对比速览

| 维度 | Amazon | Shopify | TikTok Shop |
|------|--------|---------|-------------|
| 流量来源 | 站内搜索为主 | 站外引流为主（SEO/广告/社交） | 算法推荐 + 短视频 |
| AI 核心场景 | Listing SEO、PPC 优化、Review 分析 | 产品页 SEO、Facebook/Google Ads、邮件营销 | 短视频生成、达人匹配、直播脚本 |
| 数据获取 | Seller Central 报告 | Shopify Analytics + GA4 | TikTok Seller Center |
| 竞争模式 | 同平台竞品直接对比 | 跨渠道品牌竞争 | 内容质量 + 算法推荐 |
| AI 工具生态 | Helium 10、Jungle Scout | Shopify Magic、Klaviyo AI | CapCut、GMV Max |
| 关键差异 | 平台规则驱动 | 品牌自主权高 | 内容驱动 |
