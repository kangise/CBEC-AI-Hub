# Stage 4: 数据驱动 (Analytics)

> 🎯 目标：用数据驱动决策，从"凭感觉"升级到"看数据"
> 
> 👤 适合：L3-L4 | ⏱️ 预计时间：3-6个月

---

## 4.1 数据分析思维与方法论

### 你需要掌握的

- [ ] 数据分析的基本框架（提问 → 收集 → 清洗 → 分析 → 呈现 → 行动）
- [ ] 电商核心指标体系（流量 × 转化 × 客单价 × 复购）
- [ ] 漏斗分析（Impression → Click → Add to Cart → Purchase）
- [ ] 同比/环比分析、趋势分析
- [ ] 归因分析与因果推断基础
- [ ] 数据可视化原则

### 学习资源

**课程**
- 📺 [Coursera: Google Data Analytics Professional Certificate](https://www.coursera.org/professional-certificates/google-data-analytics) — Google 数据分析认证（强烈推荐，适合零基础）
- 📺 [Coursera: Data Analysis with Python (IBM)](https://www.coursera.org/learn/data-analysis-with-python) — Python 数据分析
- 📺 [Khan Academy: Statistics & Probability](https://www.khanacademy.org/math/statistics-probability) — 统计学基础（免费）

**书籍**
- 📚 *Storytelling with Data* (Cole Nussbaumer Knaflic) — 数据可视化圣经
- 📚 *Lean Analytics* (Alistair Croll) — 精益数据分析
- 📚 *Naked Statistics* (Charles Wheelan) — 统计学入门（通俗易懂）

**视频**
- 🎬 [YouTube: Data Analytics for Beginners (Alex The Analyst)](https://www.youtube.com/@AlexTheAnalyst) — 数据分析入门
- 🎬 [YouTube: 电商数据分析方法论](https://www.youtube.com/results?search_query=电商+数据分析+方法论)

## ✅ Stage 4 能力检验

1. **数据看板** — 用 Excel 或 BI 工具搭建一个包含核心 KPI 的运营仪表板
2. **A/B 测试报告** — 设计并执行一个 Listing A/B 测试，撰写完整的测试报告
3. **财务模型** — 建立一个产品级的 P&L 模型，包含场景分析
4. **竞品分析报告** — 对你的核心竞品做一份深度分析报告（价格策略、广告策略、产品线布局）
